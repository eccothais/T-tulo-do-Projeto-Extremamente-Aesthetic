# Natural ou Fake Natty? Como Vencer na Era das IAs Generativas

## 📒 Descrição
Este vídeo apresenta uma visão rápida e envolvente do potencial e da aplicação das IAs Generativas na vida cotidiana.

## 🤖 Tecnologias Utilizadas
- IA Generativa ** [[bing.com (https://bing.com/images**] para criar a imagem;
- IA Generativa **[(https://deevid.ai/pt/my-creations)]** para criar o video.

## 🧐 Processo de Criação
- Usei o Bing para a criação da imagem da IA usando a IA.
- E após esse processo usei o DeeVid IA para a criação do vídeo.

## 🚀 Resultados
O Video descreve a IA criando "vida". A ideia foi a IA se adaptando e aprendendo cada vez mais, com o intuito ensinar pra ela mesmo sobre ela.
